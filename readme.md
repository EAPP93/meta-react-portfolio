# **Instrucciones para Clonar, Instalar y Ejecutar el Proyecto**

¡Bienvenido al proyecto! Aquí encontrarás las instrucciones necesarias para clonar el repositorio, instalar las dependencias y ejecutar el proyecto en tu entorno local.

## **Clonar el Repositorio**
Abre tu terminal o línea de comandos.

Navega hasta la ubicación donde deseas guardar el proyecto usando el comando

``` bash
cd "ruta-del-repositorio"
```

Ejecuta el siguiente comando para clonar el repositorio desde GitHub:

Copy code

``` bash
  git clone https://github.com/EAPP93/meta-react-portfolio.git
```

## **Instalar Dependencias**
Asegúrate de tener Node.js y npm (Node Package Manager) instalados. Puedes descargarlos desde https://nodejs.org/.

Navega hasta el directorio del proyecto utilizando 
``` bash
cd "tu-repositorio"
```

Una vez en el directorio del proyecto, ejecuta el siguiente comando para instalar las dependencias:

Copy code
``` bash
npm install
```
## **Ejecutar el Proyecto**
Después de instalar las dependencias, ejecuta el proyecto con el siguiente comando:

Copy code
``` bash
npm start
```

El proyecto se ejecutará y podrás acceder a él en tu navegador web visitando la dirección ```http://localhost:puerto``` (donde \"puerto\" es el número de puerto en el que se está ejecutando el proyecto, normalmente "3000").

¡Listo! Ahora deberías tener el proyecto clonado, las dependencias instaladas y el proyecto en ejecución en tu entorno local.
